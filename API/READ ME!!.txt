Anotaciones:

A veces cuando transportaba la API de casa al centro me daba un error sobre el path de algunos archivos descargados de sqlalchemy y FastAPI.
La solución mas rápida que encontré es crear un nuevo entorno y copiar y pegar la estructura y los archivos en un nuevo entorno para seguido descargar los siguientes paquetes:

pip install fastapi
pip install sqlalchemy
